Project Overview

A comprehensive SQL-based analysis of pizza sales data to uncover business insights and drive data-driven decisions for a pizza restaurant chain. This project demonstrates advanced SQL techniques applied to real-world restaurant data to optimize operations, increase revenue, and enhance customer satisfaction.

Key Features

  • Analysis of 21,351 orders and 48,621 order details to identify sales patterns
  • Four comprehensive datasets including orders, order details, pizzas, and pizza types
  • Revenue calculations, peak hour analysis, and category performance metrics
  • Advanced SQL techniques including window functions, CTEs, and ranking functions
  • Business insights for inventory management, staffing, and marketing strategies
  • Professional presentation with visual summaries and recommendations

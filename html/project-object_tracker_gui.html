Project Overview

A multipurpose object tracking system that detects and tracks multiple objects in video footage using YOLOv5 for detection and SORT (Simple Online and Realtime Tracking) algorithm for multi-object tracking. Each detected object is assigned a unique ID that remains consistent across frames. While demonstrated with football player tracking, this system is designed as a multipurpose tracking solution suitable for various applications including traffic monitoring, crowd analysis, and wildlife observation.

Football was specifically chosen as the demonstration case because it involves tracking many players (typically 22 on the field) with dynamic momentum and frequent interactions, making it an ideal showcase for the system's advanced tracking capabilities.

Key Features

Multi-Object Detection

Utilizes YOLOv5 for accurate detection of multiple objects in each video frame with high precision.

Real-time Tracking

Employs SORT algorithm for consistent tracking of objects across frames with unique ID assignment.

Enhanced Popup Window

Provides detailed view of selected objects with 400x400 pixel resolution and advanced image processing.

Multi-Digit ID Selection

Supports selection of objects with IDs beyond single digits (e.g., 21, 56, 123) with confirmation system.

Performance Optimized

Works efficiently on CPU without requiring GPU, with scalable architecture for dozens to hundreds of objects.

Multipurpose Design

Suitable for various tracking scenarios including sports analytics, traffic monitoring, and security surveillance.

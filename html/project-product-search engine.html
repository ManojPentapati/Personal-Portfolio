Project Overview

A web application that allows users to search for products across multiple e-commerce platforms simultaneously. This project focuses on creating a seamless experience for comparing products and prices from various online retailers.

Key Features

  • Search across major e-commerce platforms including Amazon, Flipkart, Meesho, Myntra, Ajio, BuyHatke, MySmartPrice, and Google Shopping
  • Automatically loads relevant product images based on search terms using Unsplash API
  • Corrects common misspellings in search queries for better results
  • Open product pages directly or copy URLs to clipboard
  • Download all generated search links as a text file

Technologies Used

HTML5 CSS3 JavaScript(ES6) Unsplash Source API for product images
